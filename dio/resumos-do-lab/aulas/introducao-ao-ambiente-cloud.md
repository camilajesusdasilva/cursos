### Versionamento de Código com Git e GitHub

A diferença de Git e GitHub:

- Git é um sistema de versionamento de código, onde criamos um repositório que pode ser local ou enviado para um armazenamento remoto;
- GitHub é um dos tipos de armazenamento remoto de repositórios mais utilizados pelas empresas atualmente.

Como utilizar o Git:

- O Git é uma plataforma utilizada pelo terminal. Utilizando os comandos que tem como ponto de partida o comando principal "git", podemos fazer coisas como:

  - Iniciar um repositório local (git init);
  - Baixar o código fonte de um repositório remoto (git clone);
  - Preparar o conteúdo para um próximo commit (git add);
  - Mostrar o status (arquivos adicionados, alterados, excluídos) do repositório (git status)
  E entre outros.

Como utilizar o GitHub:

- Utilizamos o GitHub para armazenar nossos repositórios com os projetos que fizemos, mas também utilizamos para nos apresentar ao mercado do trabalho.
Com um bom ReadMe, os recrutadores podem ficar mais interessados no nosso perfil profissional.
